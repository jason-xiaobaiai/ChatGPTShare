## 使用教程

### 使用方法：像数 1, 2, 3 一样容易

步骤 1：接入 [NextAI 用户管理中心](https://api.nextai.win) 后台创建 API Key 令牌填入

步骤 2：接口地址修改为：https://api.nextai.win

### 常见使用场景

NextAI 适用于众多 AI 应用，基本支持所有可以自定义 OpenAI API 代理的网站、应用和类库。 以下为常用的使用场景，更多 AI 使用场景可以尽情探索。

[**NextChat**](https://chat.nextai.trade)

一款功能强大的 ChatGPT 网页版，经典UI，极致简洁，推荐使用。选择 OpenAI 模型，输入接口地址和 API Key 即可开始使用。

[**BotGem**](https://botgem.com)

一款功能强大界面简洁的 ChatGPT 客户端，适配 ios 系统、Android 系统，支持 MacOS 和 Windows，注重用户体验，界面简洁舒适，推荐使用。选择 OpenAI 模型，输入接口地址和 API Key 即可开始使用。

[**沉浸式翻译**](https://immersivetranslate.com)

最受欢迎的双语对照网页翻译插件，能识别主要内容后按段落同时显示原文和译文。选择 OpenAI 模型，输入接口地址和 API Key 即可开始使用。
